我其实有很多有意义的工作，不过现在还尚且不能接受。

有时候我总之在变化，有时是“失足未必千古恨，今朝立志做新人”，有时是“能动地改造世界”，有时是“终于有一天，我们会完全站在光明中”，有时是“我一出手，还未失手”，看试手，[补天缺
](https://www.cnblogs.com/Elegia/p/17939570/make-up)。

今看昔日的文章，思想感情的传达是很到位了，但是缺少自信和力量。这曾经可能有一个经常被混淆的名字：“风骨”，其实当时经历的也少，情绪波动的也大，尤其是，还有很多未妥协的地方。如果什么时候我们知道仅仅是做到那些最基本的就已经很费力了，也许就可以省去几乎全部的担忧，而增添很多的力量，勇气，信念了。
